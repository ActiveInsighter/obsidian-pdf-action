# 机器学习复习题目第四套：机器学习综合测试详细讲解

来源：Google Drive 文件《复习题目.md》中的《机器学习综合测试 题目与解答整理》。原卷共 100 题，前 60 题为单选题，后 40 题为判断题。本讲解按原题顺序整理，不放答案速览。

## 全部题目与答案解析

### 1. 单选题（1分）

**题目：** CART算法生成的决策树是什么类型的树？

- A. 二叉树
- B. 红黑树
- C. 多叉树

**答案：A**

**解析：** CART 的全称是 Classification and Regression Tree，即分类与回归树。它每次划分都把数据分成两个子集，所以生成的是二叉树。ID3、C4.5 常见结果可以是多叉树，但 CART 的核心特点就是二叉划分。

### 2. 单选题（1分）

**题目：** 以下哪个不是常见的激活函数？

- A. tanh
- B. MSE
- C. Sigmoid
- D. ReLU

**答案：B**

**解析：** tanh、Sigmoid、ReLU 都是神经网络中常见的激活函数，用来引入非线性。MSE 是均方误差，是回归任务常用损失函数，不是激活函数。

### 3. 单选题（1分）

**题目：** 以下关于偏差-方差的说法，正确的是？

- A. 复杂模型偏差大方差小
- B. 偏差和方差总是同时增大
- C. 简单模型偏差小方差大
- D. 简单模型偏差大方差小

**答案：D**

**解析：** 简单模型表达能力弱，容易欠拟合，所以偏差大；但它对训练集扰动不太敏感，所以方差小。复杂模型通常偏差小、方差大，容易过拟合。

### 4. 单选题（1分）

**题目：** 最小二乘法的优化目标是什么？

- A. 最小化预测值与真实值之积
- B. 最小化预测值与真实值之差的平方和
- C. 最大化似然函数
- D. 最大化预测值与真实值之差

**答案：B**

**解析：** 最小二乘法的目标是让所有样本的预测误差平方和最小，即

\[
\min \sum_{i=1}^{n}(y_i-\hat y_i)^2
\]

平方可以避免正负误差相互抵消，并且会放大大误差。

### 5. 单选题（1分）

- A. 不需要训练
- B. 只能用于图像
- C. 不需要数据
- D. 可以自动提取特征

**解析：** 传统机器学习往往需要人工设计特征，例如手工提取纹理、形状、统计量。深度学习通过多层神经网络从原始数据中逐层学习特征表示，因此更强调自动特征提取。

### 7. 单选题（1分）

**题目：** 决策树信息增益计算题。某数据集共10个样本，其中6个为正类 Yes，4个为负类 No。特征A“天气”划分为：晴天5个样本，其中4个Yes、1个No；阴天5个样本，其中2个Yes、3个No。特征B“温度”划分为：高温4个样本，其中3个Yes、1个No；适中6个样本，其中3个Yes、3个No。计算整个数据集的信息熵、两个特征划分后的条件熵和信息增益，并选择第一个分裂特征。

- A. 不会做
- B. 已掌握

**答案：B**

**解析：** 原数据集正类比例为 \(0.6\)，负类比例为 \(0.4\)，所以

\[
H(D)=-0.6\log_2 0.6-0.4\log_2 0.4\approx 0.971
\]

按“天气”划分：晴天子集 \(4:1\)，熵约为 \(0.722\)；阴天子集 \(2:3\)，熵约为 \(0.971\)。因此条件熵为

\[
H(D|天气)=\frac{5}{10}\times0.722+\frac{5}{10}\times0.971\approx0.847
\]

信息增益为

\[
Gain(D,天气)=0.971-0.847\approx0.124
\]

按“温度”划分：高温子集 \(3:1\)，熵约为 \(0.811\)；适中子集 \(3:3\)，熵为 \(1\)。因此

\[
H(D|温度)=\frac{4}{10}\times0.811+\frac{6}{10}\times1\approx0.924
\]

信息增益为

\[
Gain(D,温度)=0.971-0.924\approx0.047
\]

因为“天气”的信息增益更大，所以应选择“天气”作为第一个分裂特征。

### 8. 单选题（1分）

**题目：** 支持向量机 SVM 间隔计算题：已知超平面为 \(3x_1+4x_2-10=0\)，支持向量点为 \((3,4)\)。计算该点到超平面的距离、分类间隔，并判断新点 \((1,1)\) 属于正类还是负类，默认大于0为正类。

- A. 不会做
- B. 已掌握

**答案：B**

**解析：** 点到直线 \(Ax+By+C=0\) 的距离为

\[
d=\frac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}
\]

代入 \(A=3,B=4,C=-10,(x_0,y_0)=(3,4)\)：

\[
d=\frac{|3\times3+4\times4-10|}{\sqrt{3^2+4^2}}=\frac{15}{5}=3
\]

如果题目把 margin 理解为两侧间隔宽度，则分类间隔为 \(2d=6\)；如果按单侧几何间隔，则为 \(3\)。本题通常按“分类间隔宽度”写 \(6\)。新点 \((1,1)\) 代入判别函数：

\[
f(1,1)=3\times1+4\times1-10=-3<0
\]

所以新点属于负类。

### 9. 单选题（1分）

**题目：** 拉普拉斯平滑的目的是解决什么问题？

- A. 欠拟合
- B. 零概率问题
- C. 计算复杂度高
- D. 过拟合

**答案：B**

**解析：** 在朴素贝叶斯中，如果某个类别下某个特征值在训练集中从未出现，则概率会被估计为 \(0\)，导致整个后验概率被乘成 \(0\)。拉普拉斯平滑通过加小常数避免零概率。

### 12. 单选题（1分）

- C. 准确率低

### 13. 单选题（1分）

- C. 控制学习率
- D. 控制核函数类型

### 18. 单选题（1分）

**题目：** PCA 计算题。给定二维数据点 \((2,4),(3,5),(4,6),(5,7),(6,8)\)，手动执行 PCA 降为一维：中心化、计算协方差矩阵和特征值，选择最大特征值对应方向，并把第一个样本投影到该方向。

- A. 不会做
- B. 已掌握

**答案：B**

**解析：** 两个维度的均值分别为

\[
\bar x=4,\quad \bar y=6
\]

中心化后的数据为

\[
(-2,-2),(-1,-1),(0,0),(1,1),(2,2)
\]

设中心化矩阵为 \(X\)，题目给出 \(C=\frac{1}{n}X^TX\)，则

\[
X^TX=\begin{bmatrix}10&10\\10&10\end{bmatrix},\quad
C=\begin{bmatrix}2&2\\2&2\end{bmatrix}
\]

该矩阵特征值为 \(4\) 和 \(0\)，最大特征值 \(4\) 对应的单位特征向量可取

\[
\frac{1}{\sqrt2}(1,1)^T
\]

第一个样本中心化后为 \((-2,-2)\)，投影值为

\[
(-2,-2)\cdot\frac{1}{\sqrt2}(1,1)^T=-\frac{4}{\sqrt2}=-2\sqrt2
\]

### 22. 单选题（1分）

**题目：** 贝叶斯定理中，后验概率正比于什么？

- A. 证据
- B. 似然
- C. 先验概率乘以似然
- D. 先验概率

**答案：C**

**解析：** 贝叶斯定理为

\[
P(C|X)=\frac{P(X|C)P(C)}{P(X)}
\]

其中 \(P(X)\) 对所有类别比较时相同，所以后验概率正比于 \(P(C)P(X|C)\)，即先验概率乘以似然。

### 26. 单选题（1分）

**题目：** 精确率（Precision）的计算公式是？

- A. TP/(TP+FP)
- B. (TP+TN)/总样本数
- C. TP/(TP+FN)
- D. FP/(FP+TN)

**答案：A**

**解析：** 精确率关注“预测为正的样本中有多少是真的正”，公式为

\[
Precision=\frac{TP}{TP+FP}
\]

召回率才是 \(\frac{TP}{TP+FN}\)，准确率是 \(\frac{TP+TN}{总样本数}\)。

### 28. 单选题（1分）

**题目：** 分类模型评价指标计算题。混淆矩阵为：实际正类中预测正类45、预测负类5；实际负类中预测正类15、预测负类35。计算 Accuracy、Precision、Recall、F1，并说明何时关注精确率或召回率。

- A. 不会做
- B. 已掌握

**答案：B**

**解析：** 根据题目可得：

\[
TP=45,\quad FN=5,\quad FP=15,\quad TN=35
\]

总样本数为 \(45+5+15+35=100\)。准确率为

\[
Accuracy=\frac{TP+TN}{100}=\frac{45+35}{100}=0.80
\]

精确率为

\[
Precision=\frac{TP}{TP+FP}=\frac{45}{45+15}=0.75
\]

召回率为

\[
Recall=\frac{TP}{TP+FN}=\frac{45}{45+5}=0.90
\]

F1 分数为

\[
F1=\frac{2PR}{P+R}=\frac{2\times0.75\times0.90}{0.75+0.90}\approx0.818
\]

如果误报代价高，应优先关注精确率，例如垃圾邮件过滤把正常邮件错判为垃圾邮件。若漏报代价高，应优先关注召回率，例如疾病筛查、欺诈检测。

### 34. 单选题（1分）

**题目：** KNN计算题。训练数据为：1号 \((0,2)\) 负类，2号 \((4,3)\) 正类，3号 \((2,5)\) 正类，4号 \((6,6)\) 负类，5号 \((5,4)\) 负类。待分类样本 \(A(3,4)\)。计算欧氏距离；当 \(K=3\) 时分类；当 \(K=1\) 时分类并说明影响。

- A. 不会做
- B. 已掌握

**答案：B**

**解析：** 欧氏距离公式为

\[
d=\sqrt{(x_1-x_2)^2+(y_1-y_2)^2}
\]

样本 \(A(3,4)\) 到各训练样本距离为：

- 到1号：\(\sqrt{(3-0)^2+(4-2)^2}=\sqrt{13}\approx3.6\)，负类；
- 到2号：\(\sqrt{(3-4)^2+(4-3)^2}=\sqrt2\approx1.4\)，正类；
- 到3号：\(\sqrt{(3-2)^2+(4-5)^2}=\sqrt2\approx1.4\)，正类；
- 到4号：\(\sqrt{(3-6)^2+(4-6)^2}=\sqrt{13}\approx3.6\)，负类；
- 到5号：\(\sqrt{(3-5)^2+(4-4)^2}=2\)，负类。

当 \(K=3\) 时，最近三个邻居为2号、3号、5号，其中正类2个、负类1个，所以分类为正类。当 \(K=1\) 时，最近距离为2号和3号并列，二者都是正类，所以仍为正类。\(K\) 越小越容易受局部样本影响，\(K\) 越大越平滑但可能欠拟合。

### 58. 单选题（1分）

**题目：** 线性回归与梯度下降计算题。模型 \(\hat y=wx+b\)，当前 \(w=1.0,b=0.5\)，学习率 \(\eta=0.1\)，训练样本 \((x,y)=(2,5)\)，损失函数 \(L=\frac{1}{2}(\hat y-y)^2\)。计算预测值和损失，求偏导并更新参数，再判断新预测值与真实值差距变化。

- A. 已掌握
- B. 不会做

**答案：A**

**解析：** 当前预测值为

\[
\hat y=1.0\times2+0.5=2.5
\]

损失为

\[
L=\frac{1}{2}(2.5-5)^2=\frac{1}{2}\times6.25=3.125
\]

对 \(w\) 和 \(b\) 求偏导：

\[
\frac{\partial L}{\partial w}=(\hat y-y)x=(2.5-5)\times2=-5
\]

\[
\frac{\partial L}{\partial b}=\hat y-y=2.5-5=-2.5
\]

梯度下降更新：

\[
w'=w-\eta\frac{\partial L}{\partial w}=1.0-0.1\times(-5)=1.5
\]

\[
b'=b-\eta\frac{\partial L}{\partial b}=0.5-0.1\times(-2.5)=0.75
\]

更新后的预测值为

\[
\hat y'=1.5\times2+0.75=3.75
\]

原来与真实值差距为 \(2.5\)，更新后差距为 \(1.25\)，所以差距缩小了。

**题目：** 先验概率是根据新信息修正后的概率。

**答案：错误**

**解析：** 先验概率是在观察新证据之前的概率。根据新信息修正后的概率叫后验概率。

### 80. 判断题（1分）

**题目：** RBF核是SVM中最常用的核函数之一。

**答案：正确**

**解析：** RBF 核又叫高斯核，能够处理非线性分类问题，是 SVM 中非常常用的核函数。

### 81. 判断题（1分）

**题目：** 学习率是梯度下降法中最重要的超参数。

**答案：正确**

**解析：** 学习率决定参数每次更新的步长。太小收敛慢，太大可能震荡或发散，因此它是梯度下降中非常关键的超参数。

### 82. 判断题（1分）

**题目：** PCA可以保留原始数据的全部信息。

**答案：错误**

**解析：** 如果保留全部主成分，理论上可以不丢信息；但 PCA 常用于降维，保留部分主成分时会丢失低方差方向上的信息。因此不能说 PCA 一定保留全部信息。

### 83. 判断题（1分）

**题目：** KNN算法的时间复杂度与训练样本数量无关。

**答案：错误**

**解析：** KNN 预测时需要计算待测样本到训练样本的距离，训练样本越多，计算量越大，所以时间复杂度与训练样本数量有关。

### 84. 判断题（1分）

**题目：** PCA是一种无监督学习方法。

**答案：正确**

**解析：** PCA 不使用类别标签，只根据数据本身的方差结构寻找主成分方向，因此属于无监督学习。

### 85. 判断题（1分）

**题目：** K-means对噪声和孤立点数据不敏感。

**答案：错误**

**解析：** K-means 使用均值作为质心，均值容易被极端点拉偏，因此对噪声和孤立点比较敏感。

### 86. 判断题（1分）

**题目：** Bagging算法的代表是随机森林。

**答案：正确**

**解析：** 随机森林通过 bootstrap 抽样训练多棵决策树，再投票或平均，正是 Bagging 思想的典型代表。

### 87. 判断题（1分）

**题目：** 聚类的目标是簇内相似度高、簇间相似度低。

**答案：正确**

**解析：** 聚类希望同一簇内部样本尽可能相似，不同簇之间尽可能不同，这是聚类任务的基本目标。

### 88. 判断题（1分）

**题目：** AUC=0.5表示分类器效果等同于随机猜测。

**答案：正确**

**解析：** ROC 曲线下的面积 AUC 为0.5时，模型区分正负样本的能力大致相当于随机猜测。

### 89. 判断题（1分）

**题目：** PCA假设数据之间的关系是线性的。

**答案：正确**

**解析：** PCA 是线性降维方法，用线性组合构造主成分，因此主要刻画线性相关结构。非线性结构通常需要核 PCA、ISOMAP 等方法。

### 90. 判断题（1分）

**题目：** 最小二乘法需要计算矩阵的逆。

**答案：正确**

**解析：** 题库答案为正确。线性回归最小二乘的正规方程常写为

\[
\theta=(X^TX)^{-1}X^Ty
\]

这里需要矩阵逆。不过实际工程中也常用伪逆、QR分解或梯度下降来避免直接求逆。考试按正规方程理解即可。

### 91. 判断题（1分）

**题目：** KNN中的K值通常取偶数。

**答案：错误**

**解析：** 分类任务中 \(K\) 通常取奇数，以减少投票平局的可能。若类别数较多，还需结合交叉验证选择合适 \(K\)。

### 92. 判断题（1分）

**题目：** KNN需要进行显式的模型训练过程。

**答案：错误**

**解析：** KNN 是懒惰学习算法，训练阶段主要保存训练数据，预测时才进行距离计算和投票，因此没有复杂的显式训练过程。

### 93. 判断题（1分）

**题目：** C5.0是C4.5的商用改进版本。

**答案：正确**

**解析：** C5.0 可以看作 C4.5 的改进版本，在效率、内存、规则生成等方面有所优化，常被称为商用改进版本。

### 94. 判断题（1分）

**题目：** 后剪枝的训练时间比预剪枝长。

**答案：正确**

**解析：** 后剪枝要先生成较完整的树，再进行剪枝评估；预剪枝在生长过程中提前停止。因此后剪枝通常训练时间更长。

### 95. 判断题（1分）

**题目：** 核技巧可以在不增加计算复杂度的情况下实现高维映射。

**答案：正确**

**解析：** 题库答案为正确。准确理解是：核技巧不需要显式构造高维特征向量，而是通过核函数直接计算高维空间中的内积，从而避免显式高维映射带来的巨大计算量。实际计算核函数本身仍有成本，但比显式映射更高效。

### 96. 判断题（1分）

**题目：** SVM被设计只能用于二分类问题，但可以通过一对多等策略扩展到多分类。

**答案：正确**

**解析：** 标准 SVM 原始形式是二分类模型。多分类可通过一对多 OvR、一对一 OvO 等策略组合多个二分类器实现。

### 97. 判断题（1分）

**题目：** 半监督学习同时使用有标签和无标签的数据。

**答案：正确**

**解析：** 半监督学习利用少量有标签数据和大量无标签数据训练模型，适合标签昂贵但无标签数据充足的场景。

### 98. 判断题（1分）

**题目：** K-means每次运行的结果一定相同。

**答案：错误**

**解析：** K-means 对初始质心敏感，初始质心不同可能得到不同聚类结果。因此实际常多次运行并选择较优结果。

### 99. 判断题（1分）

**题目：** CART算法生成的决策树是多叉树。

**答案：错误**

**解析：** CART 每次二分，生成二叉树，不是多叉树。这与第1题一致。

### 100. 判断题（1分）

**题目：** 增加训练数据量可以降低方差。

**答案：正确**

**解析：** 更多训练数据通常能让模型对单个样本扰动不那么敏感，从而降低方差、缓解过拟合。尤其对复杂模型而言，增加数据量是提升泛化能力的重要方式。
