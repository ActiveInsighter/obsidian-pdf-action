## 全部题目与答案解析

### 1. 单选题（1分）

**题目：** 贝叶斯定理中，后验概率正比于什么？

- A. 先验概率
- B. 证据
- C. 先验概率乘以似然
- D. 似然

**答案：C**

**解析：** 贝叶斯定理可写为 \(P(Y|X)=\frac{P(X|Y)P(Y)}{P(X)}\)。其中 \(P(Y)\) 是先验概率，\(P(X|Y)\) 是似然，\(P(X)\) 是证据或归一化因子。所以后验概率与“先验概率 \(\times\) 似然”成正比。

### 2. 单选题（1分）

**题目：** Boosting主要降低模型的什么？

- A. 噪声
- B. 方差
- C. 训练误差
- D. 偏差

**答案：D**

**解析：** Boosting 是串行训练思想：后一轮学习器重点关注前一轮分错的样本，从而一步步修正模型的系统性错误。系统性错误通常对应偏差，所以 Boosting 主要用于降低偏差；而 Bagging 更典型地降低方差。

### 3. 单选题（1分）

**题目：** 朴素贝叶斯中的"朴素"是指什么？

- A. 只能处理少量特征
- B. 假设特征之间相互独立
- C. 算法简单
- D. 准确率低

**答案：B**

**解析：** “朴素”不是说算法低级，而是指条件独立假设：在类别给定时，各个特征之间相互独立。这个假设使联合概率可以拆成多个条件概率的乘积，计算会大大简化。

### 4. 单选题（1分）

**题目：** K-近邻算法（KNN）计算题 已知训练数据集如下（2个特征），以及一个待分类样本 A(3, 4)： 请回答以下问题： （1）（4分）计算待分类样本 A 到每个训练样本的欧氏距离（写出计算过程，结果可保留根号或开方取一位小数）。 （2）（3分）若 K=3，根据最近的3个邻居，使用多数投票法判断样本 A 的类别。 （3）（3分）若 K=1，样本 A 的分类结果是什么？这说明 K 值大小对分类结果有何影响？

- A. 不会做
- B. 已掌握

**答案：B**

**解析：** 本题原文件没有保留训练样本表，所以不能算出唯一的数值分类结果；原卷这题实际是“是否已掌握”的自评题，答案为 B。真正做 KNN 计算时，先用欧氏距离 \(d=\sqrt{(x_1-a_1)^2+(x_2-a_2)^2}\) 算待分类点 \(A(3,4)\) 到所有训练点的距离，再按距离从小到大取最近的 \(K\) 个邻居。\(K=3\) 时用多数投票决定类别；\(K=1\) 时直接取最近邻的类别，因此对噪声和偶然样本更敏感。

### 5. 单选题（1分）

**题目：** 朴素贝叶斯分类器属于什么类型的模型？

- A. 聚类模型
- B. 回归模型
- C. 判别模型
- D. 生成模型

**答案：D**

**解析：** 朴素贝叶斯会先学习每个类别的先验概率 \(P(y)\) 和条件概率 \(P(x_i|y)\)，再用贝叶斯公式生成后验概率进行分类。它建模的是“类别如何生成特征”，所以属于生成模型，不是直接学习分类边界的判别模型。

### 6. 单选题（1分）

**题目：** 逻辑回归使用的激活函数是？

- A. ReLU
- B. tanh
- C. Softmax
- D. Sigmoid

**答案：D**

**解析：** 逻辑回归用于二分类时，把线性输出 \(z=w^Tx+b\) 输入 Sigmoid 函数，得到 \(0\) 到 \(1\) 之间的概率：\(\sigma(z)=\frac{1}{1+e^{-z}}\)。ReLU、tanh 常见于神经网络，Softmax 常用于多分类输出层。

### 7. 单选题（1分）

**题目：** SVM中参数C的作用是什么？

- A. 控制对误分类的惩罚程度
- B. 控制核函数类型
- C. 控制迭代次数
- D. 控制学习率

**答案：A**

**解析：** SVM 中的 \(C\) 是软间隔惩罚参数，用来控制“间隔尽量大”和“误分类尽量少”之间的权衡。\(C\) 越大，对误分类惩罚越重，模型更倾向于把训练样本分对；\(C\) 越小，允许更多误分类，间隔更宽，泛化可能更稳。

### 8. 单选题（1分）

**题目：** 主成分分析（PCA）计算题： 给定以下5个二维数据点： (2, 4)，(3, 5)，(4, 6)，(5, 7)，(6, 8) 请手动执行PCA算法，将其降为一维： （1）（3分）对数据进行零均值化（即每个维度减去该维度的均值），写出中心化后的数据。 （2）（4分）计算协方差矩阵（2x2），并求其特征值。（提示：协方差矩阵为 C = (1/n) * X^T*X） （3）（3分）选择最大特征值对应的特征向量作为投影方向，将第一个样本 (2, 4) 的中心化数据投影到该方向上，写出降维后的值。

- A. 已掌握
- B. 不会做

**答案：A**

**解析：** 两个维度均值分别为 \(4\) 和 \(6\)，中心化后为 \((-2,-2),(-1,-1),(0,0),(1,1),(2,2)\)。按题目提示使用 \(C=\frac{1}{n}X^TX\)，可得 \(C=\begin{bmatrix}2&2\\2&2\end{bmatrix}\)，特征值为 \(4\) 和 \(0\)，最大特征值对应方向可取 \(\frac{1}{\sqrt2}(1,1)\)。第一个样本中心化后为 \((-2,-2)\)，投影为 \((-2,-2)\cdot\frac{1}{\sqrt2}(1,1)=-2\sqrt2\)。原卷为“是否已掌握”自评题，答案为 A。

### 9. 单选题（1分）

**题目：** 高斯朴素贝叶斯假设特征服从什么分布？

- A. 均匀分布
- B. 正态分布
- C. 泊松分布
- D. 二项分布

**答案：B**

**解析：** 高斯朴素贝叶斯适用于连续特征，并假设每个类别条件下的特征服从正态分布。也就是用均值和方差描述 \(P(x_i|y)\)，再代入贝叶斯公式分类。

### 10. 单选题（1分）

**题目：** 最小二乘法的优化目标是什么？

- A. 最小化预测值与真实值之差的平方和
- B. 最大化预测值与真实值之差
- C. 最大化似然函数
- D. 最小化预测值与真实值之积

**答案：A**

**解析：** 最小二乘法的核心是让预测值与真实值的残差平方和最小：\(\sum_i(\hat y_i-y_i)^2\)。平方能避免正负误差相互抵消，同时会对大误差给予更重惩罚。

### 11. 单选题（1分）

**题目：** 距离分类超平面最近的训练样本点称为？

- A. 噪声点
- B. 异常值
- C. 中心点
- D. 支持向量

**答案：D**

**解析：** SVM 的分类边界由离超平面最近的一部分样本点决定，这些点就叫支持向量。移动非支持向量通常不会改变最终超平面，但移动支持向量可能直接改变间隔和边界。

### 12. 单选题（1分）

**题目：** 以下哪个不是常见的激活函数？

- A. MSE
- B. tanh
- C. Sigmoid
- D. ReLU

**答案：A**

**解析：** MSE 是均方误差，通常作为回归损失函数，不是激活函数。Sigmoid、tanh、ReLU 都会把神经元线性输出进行非线性变换，因此属于常见激活函数。

### 13. 单选题（1分）

**题目：** 标准差标准化（Z-score）将数据转化为什么分布？

- A. 范围为[0,1]
- B. 均值为0，方差为1
- C. 均值为1，方差为0
- D. 范围为[-1,1]

**答案：B**

**解析：** Z-score 标准化通常写作 \(x'=\frac{x-\mu}{\sigma}\)。变换后数据均值变为 \(0\)，标准差变为 \(1\)，因此方差为 \(1\)；它不会保证数据落在固定区间内。

### 16. 单选题（1分）

**题目：** 分类任务常用的损失函数是？

### 18. 单选题（1分）

**题目：** 后剪枝相比预剪枝的特点是？

- A. 训练时间更短
- B. 欠拟合风险更大
- C. 泛化性能往往更优
- D. 模型更简单

**答案：C**

**解析：** 预剪枝是在树生长过程中提前停止，容易过早停止而欠拟合；后剪枝是先让树充分生长，再从下往上剪掉对泛化无益的分支。因为后剪枝有完整树作为基础，通常泛化性能更好，但训练和验证成本更高。

### 19. 单选题（1分）

- A. 松弛变量
- B. 正则化
- C. 核函数

**答案：A**

**解析：** 软间隔 SVM 允许少量样本落入间隔内甚至被错分，为此引入松弛变量 \(\xi_i\)。松弛变量衡量样本违反间隔约束的程度，再通过惩罚参数 \(C\) 控制惩罚强弱。

### 20. 单选题（1分）

**题目：** 集成学习——多分类器融合计算题 对一个三分类问题（类别A、B、C），使用5个基分类器进行预测，结果如下： 请回答以下问题： （1）使用绝对多数投票法（Majority Voting），最终分类结果是什么？ （2）简述 Bagging 和 Boosting 两种集成策略的主要区别。

- A. 已掌握
- B. 不会做

**答案：A**

**解析：** 本题原文件没有保留 5 个基分类器的具体预测表，所以不能确定唯一的多数投票类别；原卷这题是“是否已掌握”的自评题，答案为 A。做题方法是统计 5 个预测中 A、B、C 各出现几次，出现次数最多的类别就是绝对多数投票结果。Bagging 是并行训练多个基学习器，主要降低方差；Boosting 是串行训练，后续模型更关注前面分错的样本，主要降低偏差。

### 21. 单选题（1分）

**题目：** Bagging主要降低模型的什么？

- A. 噪声
- B. 训练误差
- C. 方差
- D. 偏差

**答案：C**

**解析：** Bagging 通过 bootstrap 抽样构造多个训练子集，让多个基学习器并行训练后再投票或平均。多个模型的随机误差可以相互抵消，所以 Bagging 的主要作用是降低方差，典型代表是随机森林。

### 23. 单选题（1分）

**题目：** K折交叉验证中，每次取多少份作为测试集？

- A. K-1份
- B. K份
- C. K/2份
- D. 1份

**答案：D**

### 24. 单选题（1分）

**题目：** 奥卡姆剃刀原则的含义是？

- A. 选择特征最多的模型
- B. 选择误差最大的模型
- C. 在同等条件下选择简单的模型

### 27. 单选题（1分）

**题目：** 随机森林的两个「随机」是指什么？

- A. 随机权重和随机偏置
- B. 随机选择算法和随机选择数据
- C. 样本随机采样和特征随机选择

**答案：C**

### 28. 单选题（1分）

**题目：** K-Means 聚类计算题： 给定以下一维数据点：{1, 2, 7, 8, 9}，设 K=2，初始质心为 u1=1，u2=9。 请手动执行 K-Means 算法： （1）（4分）执行第1次迭代：将每个数据点分配到最近的质心所属的簇，写出簇C1和C2的成员。 （2）（3分）重新计算每个簇的新质心 u1" 和 u2"。 （3）（3分）执行第2次迭代：根据新质心重新分配数据点，并判断是否收敛。

- A. 已掌握
- B. 不会做

**答案：A**

**解析：** 第一次迭代中，点 \(1,2\) 离 \(u_1=1\) 更近，点 \(7,8,9\) 离 \(u_2=9\) 更近，所以 \(C_1=\{1,2\}\)，\(C_2=\{7,8,9\}\)。新质心为 \(u_1'=1.5\)，\(u_2'=8\)。第二次按 \(1.5\) 和 \(8\) 重新分配，簇仍为 \(\{1,2\}\) 与 \(\{7,8,9\}\)，质心不再变化，算法收敛。原卷为自评题，答案为 A。

### 29. 单选题（1分）

**题目：** KNN属于哪种类型的学习算法？

- A. 无监督学习
- B. 强化学习
- C. 懒惰学习（Lazy Learning）
- D. 急切学习（Eager Learning）

**答案：C**

**解析：** KNN 几乎不在训练阶段显式学习参数，而是把训练样本保存下来，预测时再计算距离并投票。因此它属于懒惰学习；训练快，但预测时计算量较大。

### 30. 单选题（1分）

**题目：** 拉普拉斯平滑的目的是解决什么问题？

- A. 计算复杂度高
- B. 过拟合
- C. 零概率问题
- D. 欠拟合

**答案：C**

**解析：** 朴素贝叶斯中如果某个特征值在某类别下从未出现，条件概率可能为 \(0\)，导致整个后验概率乘积为 \(0\)。拉普拉斯平滑通过加平滑项避免零概率问题。

### 31. 单选题（1分）

**题目：** 神经网络前向传播与梯度下降计算题： 某单层神经网络有2个输入神经元和1个输出神经元。 输入向量 x = [x1, x2] = [2, 1]，权重向量为 w = [w1, w2] = [0.5, -0.5]，偏置 b = 0.5。 激活函数为 Sigmoid：sigma(z) = 1/(1 + e^(-z))，其导数为 sigma"(z) = sigma(z)*(1 - sigma(z))。 请回答以下问题： 若真实标签为 t = 1，损失函数为均方误差 L = 0.5*(y - t)^2，求梯度 dL/dw1。（写出计算过程，结果保留四位小数） 提示：使用链式法则 dL/dw1 = (dL/dy) * (dy/dz) * (dz/dw1) 其中 dL/dy = (y - t)，dy/dz = sigma(z)*(1 - sigma(z))，dz/dw1 = x1

- A. 不会做
- B. 已掌握

**答案：B**

**题目：** 监督学习的训练数据不需要标签。

**答案：错误**

**解析：** 监督学习必须使用带标签数据，例如样本对应的类别或连续目标值。没有标签的数据通常用于无监督学习或半监督学习。

### 76. 判断题（1分）

**题目：** 朴素贝叶斯适合增量式训练。

**答案：正确**

**解析：** 朴素贝叶斯参数通常是类别计数和特征条件计数，新增数据到来时可以更新这些统计量。因为不需要完全重新训练复杂模型，所以适合增量式训练。

### 77. 判断题（1分）

**题目：** 决策树构建一般采用全局最优策略。

**答案：错误**

**解析：** 决策树构建通常采用贪心策略，每一步选择当前看来最优的划分属性或阈值。这是局部最优，不保证得到全局最优树。

### 78. 判断题（1分）

**题目：** 核技巧可以在不增加计算复杂度的情况下实现高维映射。

**答案：正确**

**解析：** 核技巧通过核函数直接计算高维空间中的内积，避免显式构造高维特征向量。严格说它不是让所有计算都没有代价，而是避免显式映射带来的巨大复杂度；按考试表述，本题判断为正确。

### 79. 判断题（1分）

**题目：** 剪枝会降低在训练集上的准确率。

**答案：正确**

**解析：** 剪枝会删除一些能拟合训练集细节的分支，所以训练集准确率可能下降。但它通常能减少过拟合，从而提升验证集或测试集表现。

### 80. 判断题（1分）

**题目：** KNN需要进行显式的模型训练过程。

**答案：错误**

**解析：** KNN 属于懒惰学习，训练阶段主要是保存样本，并不显式学习参数化模型。真正的计算主要发生在预测阶段，即计算距离和投票。

### 81. 判断题（1分）

**题目：** 增加训练数据量可以降低方差。

**答案：正确**

**解析：** 增加训练数据量通常能让模型看到更多样本变化，降低对特定训练集的偶然依赖。因此它有助于降低方差，缓解过拟合。

### 82. 判断题（1分）

**题目：** SVM只能用于二分类问题。

**答案：正确**

**解析：** 按本课程考试口径，基础 SVM 本质上是二分类最大间隔分类器，因此本题判断为正确。实际应用中，多分类 SVM 常通过一对多或一对一策略扩展得到，所以复习时要区分“基本模型”和“工程扩展”。

### 83. 判断题（1分）

**题目：** 预剪枝可能导致欠拟合。

**答案：正确**

**解析：** 预剪枝在树还没完全生长时就提前停止，若停止条件过强，模型可能没有学到足够结构。这样会导致欠拟合，所以本题为正确。

### 84. 判断题（1分）

**题目：** C4.5算法能够处理不完整数据。

**答案：正确**

**解析：** C4.5 相比 ID3 有多项改进，包括使用信息增益率、能处理连续属性，并能在一定程度上处理缺失值或不完整数据。因此本题判断为正确。

### 85. 判断题（1分）

**题目：** 混淆矩阵只适用于二分类问题。

**答案：错误**

**解析：** 混淆矩阵不仅能用于二分类，也能用于多分类。多分类混淆矩阵的行列分别表示真实类别与预测类别，各单元记录对应数量。

### 86. 判断题（1分）

**题目：** 最小二乘法需要计算矩阵的逆。

**答案：正确**

**解析：** 按正规方程求最小二乘解时有 \(\hat w=(X^TX)^{-1}X^Ty\)，因此需要矩阵求逆或等价的线性方程求解。实际工程中也可用 QR 分解、SVD、梯度下降等方式避免直接求逆，但本题按正规方程口径判断为正确。

### 87. 判断题（1分）

**题目：** 硬间隔SVM允许部分样本分类错误。

**答案：错误**

**解析：** 硬间隔 SVM 要求所有样本被正确分类，并且满足间隔约束，不允许分类错误。允许部分错误的是软间隔 SVM。

### 88. 判断题（1分）

**题目：** 先验概率是根据新信息修正后的概率。

**答案：错误**

**解析：** 先验概率是在观察新证据之前已有的概率。根据新信息修正后的概率叫后验概率，所以本题说法错误。

### 89. 判断题（1分）

**题目：** 精确率和召回率总是同时增大。

**答案：错误**

**解析：** 精确率和召回率常存在权衡关系。提高判定为正类的阈值，精确率可能上升但召回率可能下降；降低阈值则可能相反，所以二者不总是同时增大。

### 90. 判断题（1分）

**题目：** 朴素贝叶斯常用于垃圾邮件过滤。

**答案：正确**

**解析：** 垃圾邮件过滤是朴素贝叶斯的经典应用。邮件中的词语可作为特征，通过学习垃圾邮件和正常邮件的词频分布来计算邮件属于垃圾邮件的概率。

### 91. 判断题（1分）

**题目：** 特征提取会增加数据的冗余。

**答案：错误**

**解析：** 特征提取的目的通常是提取更有代表性的低维信息，减少冗余和噪声。若反而增加冗余，就违背了特征提取的主要目标。

### 92. 判断题（1分）

**题目：** 欧氏距离适用于基于坐标的数据度量。

**答案：正确**

**解析：** 欧氏距离衡量坐标空间中两点的直线距离，因此适合数值坐标型数据。使用前通常要考虑标准化，否则尺度大的特征会主导距离。

### 93. 判断题（1分）

**题目：** 朴素贝叶斯的分类效率不稳定。

**答案：错误**

**解析：** 朴素贝叶斯训练和预测都比较高效，分类效率通常较稳定。它的主要问题更多是独立性假设过强，而不是分类效率不稳定。

### 94. 判断题（1分）

**题目：** Sigmoid函数的值域是(-1, 1)。

**答案：错误**

**解析：** Sigmoid 函数值域是 \((0,1)\)，不是 \((-1,1)\)。值域为 \((-1,1)\) 的常见激活函数是 tanh。

### 95. 判断题（1分）

**题目：** 信息增益倾向于选择取值较多的属性。

**答案：正确**

**解析：** 信息增益偏向取值较多的属性，因为取值越多越容易把训练样本划分得更纯。C4.5 使用信息增益率就是为了解决 ID3 的这个偏好。

### 96. 判断题（1分）

**题目：** 半监督学习同时使用有标签和无标签的数据。

**答案：正确**

**解析：** 半监督学习同时利用少量有标签数据和大量无标签数据。它适合标签获取成本高、但无标签样本容易收集的场景。

### 97. 判断题（1分）

**题目：** 后剪枝的训练时间比预剪枝长。

**答案：正确**

**解析：** 后剪枝需要先生成较完整甚至完全生长的树，然后再利用验证集或准则进行剪枝。相比预剪枝，它训练过程更长，但泛化效果往往更好。

### 98. 判断题（1分）

**题目：** 交叉熵损失函数适合回归任务。

**答案：错误**

**解析：** 交叉熵主要适合分类任务，衡量预测概率分布和真实类别分布的差异。回归任务更常用均方误差、平均绝对误差、Huber 损失等。

### 99. 判断题（1分）

**题目：** 强化学习强调基于环境行动以取得最大化的预期利益。

**答案：正确**

**解析：** 强化学习中，智能体与环境交互，执行动作，获得奖励，并学习能最大化长期期望回报的策略。所以本题说法正确。

### 100. 判断题（1分）

**题目：** 朴素贝叶斯属于判别模型。

**答案：错误**

**解析：** 朴素贝叶斯学习的是 \(P(y)\) 和 \(P(x|y)\)，再由贝叶斯公式得到 \(P(y|x)\)，属于生成模型。判别模型则直接学习 \(P(y|x)\) 或决策边界，例如逻辑回归、SVM。
